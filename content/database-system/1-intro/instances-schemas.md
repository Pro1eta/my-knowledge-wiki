---
title: 数据模式
aliases:
  - instance
  - schema
---
- **模式（Schema）**：数据库的总体设计，其描述了**特定数据库的逻辑结构**。

| 数据抽象 | View level      | Logical level  | Physical level           |
| ---- | --------------- | -------------- | ------------------------ |
| 中文名  | 外模式             | 逻辑模式           | 内模式/物理模式                 |
| 外文名  | external schema | logical schema | external/physical schema |

- **实例（Instance）**：数据库在特定时间点的具体内容。

>[!NOTE]
>一个类比是：schema - 类型，instance - 变量。
