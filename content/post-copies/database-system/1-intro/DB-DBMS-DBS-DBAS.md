# 定义

- **数据库（Database，DB）**：相互关联的**数据的集合**，以文件的形式存储在系统中。
- **数据库管理系统（Database management system，DBMS）**：DB + 一系列用于访问数据库中数据的程序。
- **数据库系统（Database system，DBS）**：一般来说和 DBMS 的定义一致。在本系列笔记的范畴中，两者可替换。

> [!NOTE] 其它定义方式
>**DBMS**：DB 中管理数据的某种手段/一系列访问 DB 中数据的程序。
>
>**DBS**：DB + DBMS + Users/Administers
>
>**DBAS**（Database application system，数据库应用系统）：DB + DBMS + App. prog. + Users/Administers
